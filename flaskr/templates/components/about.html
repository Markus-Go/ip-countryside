IP Countryside 2.0

IP Countryside 2.0 ist eine auf Python basierende Open-Source-Software, die Geolokationsabfragen von IP-Adressen ermöglicht. Die Version 2.0 kann im Vergleich zu ihrer auf C++ basierenden Vorgängerversion sowohl IPv4- als auch IPv6-Adressen abfragen.

Mehrwert

IP Countryside 2.0 stellt eine vollstän- dige Datenbank als Output in verschie- denen Formaten zur Verfügung. Außerdem kann IP Countryside 2.0 per Website aufgerufen werden oder per API eingebunden werden.

Datenbankformate

MaxMind ▪ JSON ▪ MySQL ▪ SQLite ▪ Yaml ▪ CSV
